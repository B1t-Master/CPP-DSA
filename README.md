## Overview

Practical exercise is designed to refresh your understanding of C++ programming fundamentals and introduce GitHub workflow for hosting and presenting your work.

## Tasks

Each question should be implemented as a separate C++ program.  
Ensure your code includes clear comments explaining the logic and flow.

### Question 1: Days to Seconds

Write a program that:

- Prompts the user to enter the number of days
- Converts the input into seconds
- Displays the result

### Question 2: Volume of a Sphere

Write a program that:

- Prompts the user to enter the radius
- Calculates the volume of a sphere
- Use the built-in C++ exponential function for calculating r^3

### Question 3: Square Area & Perimeter (Functions)

Write a program using functions that:

- Takes the side length of a square as input
- Computes:
  1. Area
  2. Perimeter
- Displays both results

### Question 4: Character Case Checker (Functions)

Write a program that:

- Accepts a character input from the user
- Determines whether it is:
  1. Uppercase
  2. Lowercase
- Uses functions to implement the logic

### Question 5: Pseudocode Conversion

Convert the following pseudocode into a C++ program:

```
BEGIN
    SET x TO 0, y TO 20
    REPEAT
        SUBTRACT 4 FROM y
        ADD 2/y TO x
        UNTIL y IS LESS THAN 6
    DISPLAY x
END
```

### Question 6: Array & Average Calculation

Write a program that:

1. Uses a loop to allow the user to input 5 values into an array
2. Calculates the average of the values
3. Displays the result
